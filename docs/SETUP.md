# GasGuard Setup Guide

## Prerequisites
- **Node.js**: v20 (managed via .nvmrc)
- **Rust**: Stable (managed via rust-toolchain.toml)
- **pnpm**: Package manager for Node.js

## Quick Start
To bootstrap the project, run:

```bash
./scripts/setup.sh
```

This will verify your tools and install all Node.js dependencies.
